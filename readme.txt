== Description ==

Just need to active this scroll to header plugin and it will automatically show on website right bottom corner.

==installation==

Please check and follow this steps to active this plugin on your WordPress website

e.g.

1. Upload this scroll to header plugin to the '/wp-content/plugins/' directory or install through the WordPress plugins directory.
2. Activate the plugin through the 'Plugins' screen in  WordPress
3. Now Check and Go to your Wordpress Site and you will see your website of right bottom corner it's added.
